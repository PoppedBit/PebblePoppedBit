# Blood Angels Watchface - Visual Design

## Layout

```
┌────────────────────────────────────┐
│                                    │
│         BLOOD ANGELS               │  <- Red text (Crimson #DC143C)
│                                    │
│                                    │
│           12:45                    │  <- White text, large bold font
│                                    │
│                                    │
│       MON DEC 22                   │  <- Gold text (#FFD700)
│                                    │
│      For Sanguinius!               │  <- Red text (Crimson #DC143C)
│                                    │
└────────────────────────────────────┘
     Black background (#000000)
```

## Color Scheme

- **Background**: Black (#000000) - Grimdark gothic atmosphere
- **Primary (Red)**: Crimson Red (#DC143C) - Blood Angels chapter color
- **Secondary (Gold)**: Gold (#FFD700) - Imperial honors and accent
- **Text**: White (#FFFFFF) - High contrast for readability

## Theme Elements

### Blood Angels Chapter
The Blood Angels are one of the 20 First Founding Space Marine Chapters, created during the Great Crusade. They are noble warriors with a tragic flaw, carrying the genetic legacy of their Primarch Sanguinius.

### Design Philosophy
- **Gothic aesthetics**: Bold, militaristic fonts
- **Chapter pride**: Prominent chapter name
- **Practical**: Easy-to-read digital time
- **Inspirational**: Chapter motto for motivation
- **Imperial dating**: Day and date display

### Typography
- Title: GOTHIC 18 BOLD
- Time: BITHAM 42 BOLD (largest, most prominent)
- Date: GOTHIC 18
- Motto: GOTHIC 14

## Features
- Automatic time updates every minute
- 12/24 hour format support (based on watch settings)
- Date display with day name abbreviation
- Static chapter elements for quick glanceability
- Low power consumption (updates only when needed)

## Compatibility
- Pebble Classic (Aplite) - Black & White
- Pebble Steel (Aplite) - Black & White  
- Pebble Time (Basalt) - Color display
- Pebble Time Round (Chalk) - Color, round display
- Pebble 2 (Diorite) - Black & White
